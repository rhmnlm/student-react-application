# Student Catalog UI

UI for Student Spring boot Rest API

## Description

UI for Student Spring boot Rest API. Some of functions available includes get
lists of students, lists of courses, register and drop courses for students.

## Getting Started

### Dependencies
* Node.js
* MySQL 5.7+
* REST APIs Application. Can be cloned from https://github.com/rhmnlm/student-spring-application

### Installing

* Clone from https://github.com/rhmnlm/product-application-sboot
* Write ``mvn clean install`` in terminal to install all dependencies
* Set up your db of choice and configure setting in spring boot application's application.yml file
* Open React app and write ```npm install``` to install all dependencies.

### Executing program

* Run the program by simply type in

for UI Application
```
npm start
```

for Spring Application
```
mvn spring-boot:run
```
